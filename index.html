Coin Tracker Project

Project Summary:

    The cryptocurrency tracking website was developed using ReactJS for creating a dynamic and responsive user interface, leveraging JavaScript for interactivity and logic. CSS was utilized to style the application, ensuring a visually appealing design. To manage global state efficiently, the project implemented React Context. React Router was used for seamless navigation and routing between pages. ChartJS was integrated to display cryptocurrency data in visually compelling charts. For form handling, the project used react-hook-form, ensuring a smooth user experience, combined with Zod for robust schema validation. Additionally, react-number-format was employed to format numerical data like cryptocurrency prices and values, enhancing readability.

Features:
  • Explore available Cryptocurrencies: View a list of cryptocurrencies, access detailed information such as current price and market cap, and stay updated with real-time market data.
  • User Acount Capabilities: Register/Login to unlock additional features like the portfolio tracker.
  • Add Transactions to your personal portfolio: Log new transactions by specifying the cryptocurrency, transaction type (buy/sell), quantity, and notes.
  • Portfolio Tracker: Access a summary of your transaction history with options to edit or delete transactions.
